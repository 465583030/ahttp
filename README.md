***v0.1 [released](https://github.com/go-aah/ahttp/releases/latest) and tagged on Mar 07, 2017***

HTTP Library built to process, manipulate value of Request and Response (headers, body, etc).

*`ahttp` developed for aah framework. However, it's an independent library, can be used separately with any `Go` language project. Feel free to use it.*

# Installation
#### Stable Version - Production Ready
```sh
# install the library
go get -u aahframework.org/ahttp.v0
```

#### Development Version - Edge
```sh
# install the development version
go get -u aahframework.org/ahttp.v0-unstable
```
